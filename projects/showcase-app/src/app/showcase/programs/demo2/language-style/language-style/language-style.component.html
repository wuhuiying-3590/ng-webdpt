語系樣式

針對不同語系進行樣式的調整
平台預設樣式為繁中，目前平台在 layout 框架已經放置各別語系相關的 class
切換到簡中增加 "zh_CN" ，英文語系則有 "en_US" 相對應的 class 在開發的時候做應用調整。
請在 /implementation/style/index.less 新增，載入 en_US.less 或 zh_CN.less 沒有另外設計樣式則不需要載入 
@import 'en_US.less';  //  英文語系微調
@import 'zh_CN.less';  //  簡中語系微調
另外也可以自己放置在作業任何需要區別語系樣式的地方，再依樣式層級設定該語系專屬的樣式。
參考: 英文語系的條件搜尋區塊 / modal 彈窗。
[ngClass]=" '' | dwLanguage "


ex: ＜form nz-form [formGroup]="validateForm" [ngClass]=" '' | dwLanguage "＞＜/form＞
目前平台有放置各別語系相關的 class 如下，再依據需求按層級往下設定即可 英文 class 選擇器 簡中 class 選擇器 用途描述 {{data.dwCommonvarName}} {{data.dwCommonInitial}} {{data.dwCommonDescription}}
